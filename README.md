## 📖 About

**King Legacy Wiki** is an independent, open-source documentation project aiming to provide accurate, verified, and up-to-date information about King Legacy on Roblox. Unlike existing wikis that rely on unverified community edits, this project prioritizes:

- **Source Verification** — Every fact is tagged with verification status and source URL
- **Freshness Tracking** — Time-sensitive content (codes, updates, values) includes last-checked dates
- **Evidence-Based Content** — Clear distinction between verified facts and community-sourced information

## 🏗️ Architecture

### Fact Verification System

All content follows a four-tier verification system:

| Status | Description | Example Sources |
|--------|-------------|-----------------|
| ✅ **Verified** | Confirmed from official sources | Roblox game page, Official Discord, Trello |
| ⚠️ **Likely but Unverified** | Multiple high-signal sources agree | Community wikis, guide sites |
| ❓ **Disputed** | Conflicting information across sources | Multiple community reports |
| 📜 **Outdated** | Confirmed historical information only | Old patch notes, archived pages |
